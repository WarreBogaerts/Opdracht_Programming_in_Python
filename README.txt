De applicatie is een Task Manager, in de applicatie kunnen er taken toegevoegd en opgevolgd worden.
Deze applicatie kan gebruikt worden als checklijst of als lijst van zaken die moeten gedaan worden.

In het SettingsFile.docx document vindt u waar u de database moet plaatsen en hoe deze moet opgebouwd worden.


Functionaliteiten
- Taken toevoegen
- Taken starten
- Taken voltooien
- Taken bewerken
- Taken verwijderen
- Taken tonen

- Taken worden opgeslaan op een database.
- Taken worden aangepast en deze aanpassing gebeurd ook in de database.
